A new snapshot of Repose has been released and is ready for download. The main change between s1 and s2 is the fix to allow for Repose to return more than one object by querying. A test case was added to detect this in the future. Check the changeset notes for more details on this particular bug.

All previous version of Repose have been marked as deprecated.
