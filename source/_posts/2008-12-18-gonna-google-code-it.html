Halogen Creative
I have finally decided to start pushing my code out to the rest of the world. Several new projects were added to Google code this weekend ( hc-php, skittle-php, halo-php, substrate-php and repose-php ) and will hopefully be fleshed out a bit more in the coming weeks. I am excited to try out Google code and see how well it works. It was a little unfortunate that "halo" and "substrate" were both project names already taken on SourceForge... apparently Google has decided that if a project exists on SourceForge they won't allow it to exist in their system. I had hoped to avoid the *-php suffix, but it seems like, "Everyone else is doing it" ( smarty-php ) so I guess it cannot hurt too badly. I hope.
