Halo is sporting a very quick and dirty example site in the newest trunk snapshot "full" builds. The example site includes the hc-php, Skittle and Substrate dependencies in the vendors/ folder so it should work out of the box. In theory. Thanks to Alvaro's help, I can verify that this has worked for at least one person!

Halo's documentation is still pretty scarce, but I did manage to throw together a quick cheat sheet for halo_HttpRequest. This class is used by the controller and is available by way of the $httpRequest variable in the view. It provides a standardized way to access HTTP Request information, including but not limited to query params, URL params and POST params.

So, I have made quite a bit of headway on the goal I set for myself on Friday. Hopefully there will be more docs by the end of the week!
