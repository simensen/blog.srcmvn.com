Ext Scheduler

Ext Scheduler is a powerful javascript scheduling tool that can be used for task planning, staffing, project management, resource allocation and more. It is very easy to setup and customize according to your business needs.

The component is written in pure JavaScript and it is based on the superb Ext JS framework but requires nothing else to run. It has no server dependencies whatsoever, meaning it will work together with any backend such as ASP, ASP.NET, Java, PHP, Ruby etc...

via ext-scheduler.com

Digging ext-scheduler! Fits a very well for a project I am currently working on. It seems to be very flexible and has been pretty easy to work with so far.
