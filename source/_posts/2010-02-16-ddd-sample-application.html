via dddsample.sourceforge.net

I was on a serious DDD kick for awhile, but I have since backed off trying to figure out all of the various pieces to make DDD work on smaller projects.

This site has some great (working?) example code that is a great reference to see how certain patterns can be implemented in the real world. For instance, if you wanted to check out how a Repository could be implemented, check out the Hibernate CargoRepository implementation. An Entity? Check out the Cargo entity implementation.
