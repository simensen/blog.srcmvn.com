A huge milestone has been passed for Repose. To the best of my ability to test it after 3:00am, Repose now supports array collections. Some unit testing has been checked in as well. I am super excited about this.

Here is a quick example configuration taking advantage of the collection feature to handle one-to-many relationships. Take a look at the sample_Project's "bugs" property to see how it is setup. In particular, pay attention to the backref definition. This specifies which property on the related sample_User class refers back to the project.

The project class only needs to define a $bugs property and treat it like an array.


Once setup properly, adding a new bug that references an existing project will automatically insert the bug into the project's bugs array. And the first time the caller references $project->bugs the datastore will be queried for all bugs that reference the project.
